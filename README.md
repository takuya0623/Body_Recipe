# Body Recipe

## サイト概要
ボディメイクに特化したレシピサイト

### サイトテーマ
食を通じての理想の身体作り

### テーマを選んだ理由
トレーニングを始めた時に増量・減量に特化した食事の情報がなかったから
### ターゲットユーザ
理想の身体づくりを目指しているトレーニング初心者

### 主な利用シーン
投稿を見てトレーニングやダイエットのモチベーションを上げる。
